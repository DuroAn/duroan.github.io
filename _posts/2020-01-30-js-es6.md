# 들어가며

* javascript es6 발표 후 class 와 Prototype 개념이 도입 되면서 java와 유사하게 변화 됨
* 이 말인 즉슨 oop 객체지향 적 코딩을 하도록 유도 되고 있다.

# class 선언

* 선언 예시

```javascript
class Rectangle {
    constructor(height, width) { // 생성자
        this.height = height;
        this.width = width;
    }
}
```
